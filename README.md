OBJETIVOS
Cognitivos
* Compreender os princípios básicos do pensamento lógico e sua importância na resolução de
problemas, de diferentes áreas, de forma computacional.

Habilidades
* Aplicar os conceitos de lógica proposicional para desenvolver soluções lógicas em algoritmos e
estruturas de controle. * Interpretar e elaborar algoritmos simples utilizando pseudocódigo como
ferramenta de representação lógica. * Elaborar algoritmos simples em linguagem C.

Atitudes
* Experimentação Tradução de uma rotina da vida cotidiana na forma de algoritmo. Experimentação e
avaliação de diferentes algoritmos de rotinas. * Resolução de problema Resolver um problema comum
do ambiente profissional, utilizando algoritmos simples escritos em pseudocódigo. Resolver um problema
comum do ambiente profissional, utilizando algoritmos simples escritos em linguagem C.
